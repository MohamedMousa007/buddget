Payment methods · v3 (card-first)
The card metaphor now runs the whole feature: a live card preview drives setup, and a card wallet is the management page. Provider browsing stays fast & scannable (popular chips + searchable dropdown, not a card-swipe), because you can't skim 60 providers as cards.
No flip. The card never turns over — last-4 lives on the front, so nothing implies a CVV. Feels secure.
Brand colour, editable. Each provider carries its 2 primary brand colours (in the catalogue); the card defaults to the primary, and a swatch row lets the user recolour.
Unified currency dropdown. One searchable control everywhere: flag + code + full name.
No country switcher. The app derives it from the user's base currency; "Popular in …" follows automatically.
Tweaks to compare. Provider icon glyph ↔ initials, browse style carousel ↔ list, popular scope user ↔ all.
